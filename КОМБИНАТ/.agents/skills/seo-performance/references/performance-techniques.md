# Performance techniques

1. Measure production build first (median of 3 Lighthouse runs).
2. Identify LCP candidate; set dimensions, format, priority.
3. Reduce render-blocking fonts/CSS.
4. Avoid layout shift from late images/fonts.
5. Analyze bundle only after budget breach.
6. Retest the same route/viewport after each change.
7. Never claim field INP from lab-only Lighthouse.
