# Review techniques

1. Compare screenshots to design principles, not pixel nostalgia alone.
2. Walk keyboard-only path for primary journey.
3. Toggle prefers-reduced-motion.
4. Stretch copy +30% and insert long tokens (email/URL).
5. Watch console + network for unexpected failures.
6. Flag AI clichés only when they conflict with approved direction.
7. Every blocking finding links to a source requirement.
