# Token schema

Define semantic tokens (not raw one-offs):

- color: bg, fg, muted, border, accent, danger, success, focus-ring
- typography: family-sans, family-mono, size-*, leading-*, tracking-*
- space: 1–12 scale
- radius, shadow, z-index policy
- container max-width + gutters per breakpoint
- motion: duration, easing, reduced-motion equivalents

Exceptions must be listed explicitly in design-system.md.
