# Accessibility Standard

## Thresholds

- Axe critical/serious: `0`
- Unexpected console errors: `0`
- Keyboard: full page operable; no traps; logical tab order; visible focus
- Touch targets: primary interactive controls ≥ 44×44 px
- Zoom 200%: functionality preserved
- Contrast: text and UI meet WCAG 2.2 AA for critical content
- Reduced motion: non-essential motion stopped or replaced; meaning preserved
- Semantics: landmarks, heading order, labels, alt policy for informative images

## Manual checks (required)

- Focus-visible on all interactive controls
- Form errors associated with fields
- Modals/menus return focus
- Decorative images marked decorative
