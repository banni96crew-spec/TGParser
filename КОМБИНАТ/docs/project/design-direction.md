# Design direction — AegisOps

**Gate G2:** accepted 2026-07-17

## Chosen concept

**Editorial Ops** — strict editorial-tech composition: strong typographic hierarchy, graphite surfaces, off-white content plane, one signal accent for CTAs/focus. Minimal cards. No glassmorphism. No decorative gradient washes. Motion is sparse and purposeful.

## Principles mapped from references

- Large display type for hero promise
- Dense but readable body measure
- Horizontal rules / whitespace instead of card grids
- Accent reserved for interactive emphasis

## Grid / type / color / image / motion (summary)

- Grid: 12-col desktop, 4-col mobile; max width ~1120px
- Type: grotesque sans; display / title / body / mono meta
- Color: graphite `#1B1E23`, off-white `#F4F5F7`, accent `#C45C26` (signal), border muted
- Image language: abstract operational diagrams, high contrast line art; no stock handshake photos
- Motion: fade/slide short for section reveal; respect reduced-motion

Rejected alternate: “Soft SaaS pastel cards” — conflicts with trust/security narrative.
